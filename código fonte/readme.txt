Este diretório contém o arquivo "single_file_test.cpp".

Este arquivo unifica todo o código do projeto (classes, pilha e testes) em um único arquivo de código-fonte. 

Ele serve especificamente para facilitar a execução prática em compiladores C++ online, evitando a necessidade de instalar compiladores locais em sua máquina.

Como usar:
1. Abra e copie todo o conteúdo do arquivo "single_file_test.cpp".
2. Acesse um compilador online como:
   - OneCompiler (https://onecompiler.com/cpp)
   - OnlineGDB (https://www.onlinegdb.com/)
   - C++ Shell (http://cpp.sh/)
3. Cole o código no painel e clique no botão "Run" / "Execute" para rodar a simulação dos autômatos diretamente no seu navegador.
